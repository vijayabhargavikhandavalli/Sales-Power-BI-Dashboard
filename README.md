# 📊 Sales Power BI Dashboard

An interactive **Sales Analytics Dashboard** built in **Microsoft Power BI** to monitor sales performance, profitability, customer behavior, and business trends through dynamic visualizations and KPIs.

## 🚀 Project Overview

This dashboard transforms raw sales data into meaningful business insights using Power BI. It helps stakeholders monitor sales performance, identify profitable products, analyze regional trends, and support data-driven decision-making.

---

## 🎯 Objectives

- Analyze overall sales performance
- Monitor profit and profit percentage
- Identify top-performing products and categories
- Compare regional sales performance
- Track monthly sales trends
- Provide interactive business insights

---

## 📌 Key Performance Indicators (KPIs)

- 💰 Total Sales
- 📈 Total Profit
- 📊 Profit Percentage
- 📦 Total Quantity Sold
- 🛒 Total Orders
- 👥 Customer Count
- 🌍 Regional Sales
- 📅 Monthly Sales Trend

---

## 📊 Dashboard Features

- Executive KPI Cards
- Sales Trend Analysis
- Profit Analysis
- Category-wise Sales
- Region-wise Performance
- Interactive Filters (Slicers)
- Dynamic Visualizations
- Drill-down Analysis
